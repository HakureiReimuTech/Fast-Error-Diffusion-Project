# Fast-Error-Diffusion-Project
需要pip install opencv-python numpy，使用方法:python d.py 001.jpg 001o.jpg<br>
快速误差扩散工具，让二值化图像看起来就像是灰度图像一样<br>
已经过性能优化，平均20sec一张图<br>
你可以随意的复制,修改,再分发该程序，我们不对因使用本程序导致的任何损失承担责任<br>
